# dataflow-manager
在许多企业或研究机构中，大量业务数据需要由不同来源提交，并经过人工审核后才能被用于分析、决策或模型训练。   现有流程通常依赖 Excel、邮件或即时通讯工具，存在 **数据分散、状态不透明、权限混乱、统计困难** 等问题。  本项目旨在构建一个 **集中式的数据管理平台**，用于规范数据提交、审核与统计流程，提高数据质量与管理效率。  -设计并实现了一个支持多角色与权限控制的数据管理平台，用于数据提交、审核与统计分析 -构建了完整的数据生命周期流程（提交 → 审核 → 统计），并通过 Dashboard 提供实时业务指标 -后端基于 Node.js + Express，前端使用 React，实现 REST API 与 MySQL 数据建模
